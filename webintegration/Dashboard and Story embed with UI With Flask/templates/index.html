What's in this project

Three Tableau dashboards and a data story, each looking at electricity consumption from a different angle.

Overview & Geographic Analysis

State-wise consumption maps, regional pie charts, and year-over-year usage trends for all 33 states.

Comparative & Ranking

Top N / Bottom N filters, regional comparisons, and monthly consumption patterns side by side.

Lockdown Impact Analysis

How COVID-19 changed electricity usage — metro city comparisons, quarterly trends, and regional breakdowns.

About this project

This project was built as part of a Tableau data visualization course. The dashboards were designed in Tableau Desktop using real electricity consumption data from India's Central Electricity Authority.

The web application uses Flask to serve the dashboard images and story video with authentication, a gallery view, and zoom/fullscreen capabilities.
