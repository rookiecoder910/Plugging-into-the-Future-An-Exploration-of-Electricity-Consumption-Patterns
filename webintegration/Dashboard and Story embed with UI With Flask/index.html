01

Prepare Your Dashboard or Story

Before publishing, ensure everything is polished, optimized, and ready for the public audience.

Verify All Sheets

Confirm that all 15+ worksheets (Top N, Bottom N, State Consumption maps, Monthwise charts, etc.) are rendering correctly with accurate data.

Test All Filters

Validate that State, Region, Year, Date, and Top N/Bottom N filters work correctly across all three dashboards.

Clean Up Unused Sheets

Remove or hide any draft/test worksheets that are not part of the final dashboards to reduce file size and clutter.

Optimize Data Source

Use Tableau extracts (.hyper) instead of live connections. Apply data source filters to exclude unnecessary columns and rows.

Check Dashboard Layout

Set proper sizing (Fixed or Automatic) for web display. Test both Desktop and Phone layouts for responsive design.

Review Story Flow

If using Tableau Story, verify story points flow logically — from overview to COVID-19 lockdown impact to regional deep-dive.

💡
Pro Tip: Tableau Public has a 25 MB file size limit. Use "Extract" instead of "Live" connections and remove hidden sheets to stay under the limit. Our electricity consumption dataset (~851 KB CSV) is well within this threshold.
02

Sign In to Tableau Public

Connect your Tableau Desktop to Tableau Public to enable cloud publishing.

A

Open the Save Dialog

In Tableau Desktop, navigate to File → Save to Tableau Public As…

This option may be grayed out if you have a live database connection. Switch to Extract first.

B

Sign In Window Appears

A login dialog will pop up asking for your Tableau Public credentials.

  • Enter your email address
  • Enter your password
  • Click "Sign In"
C

No Account? Sign Up Free

If you don't have an account, visit public.tableau.com and click "Sign Up". It's completely free.

03

Publish Your Workbook

Upload your dashboards to the cloud and make them publicly accessible.

Name Your Workbook

After signing in, a dialog asks for the workbook name. Use a descriptive title:

Plugging into the Future — Electricity Consumption Patterns (2019-2020)

Select Sheets to Publish

Choose which sheets and dashboards to include. Recommended: publish all 3 dashboards and the story.

Dashboard 1 Dashboard 2 Dashboard 3 Story

Upload Process

Tableau will extract and upload the data. A progress bar shows the upload status. This typically takes 30–60 seconds for our dataset.

✅ Published Successfully!

Your browser will automatically open showing the published dashboard on Tableau Public. You'll receive a shareable URL like:

https://public.tableau.com/views/PluggingIntoTheFuture/Dashboard1
04

Embed in Your Website

After publishing, embed interactive dashboards directly into any webpage.

1. Get the Embed Code

On Tableau Public, click the Share button (bottom-right of your viz) and copy the embed code.

2. Paste into HTML

Add the embed code to your website. Here's the format:

<div class='tableauPlaceholder' style='width:100%;height:800px;'>
  <object class='tableauViz' style='width:100%;height:100%;'>
    <param name='host_url' value='https://public.tableau.com/' />
    <param name='site_root' value='' />
    <param name='name' value='YourWorkbookName/Dashboard1' />
    <param name='tabs' value='yes' />
    <param name='toolbar' value='yes' />
    <param name='animate_transition' value='yes' />
  </object>
</div>

3. Add the JavaScript API

Include Tableau's JS API for full interactivity:

<script src='https://public.tableau.com/javascripts/api/viz_v1.js'></script>
⚠️
Important: Tableau Public workbooks are always public. Do not publish sensitive or confidential data. Our electricity consumption data is sourced from public databases, so it's safe to publish.
